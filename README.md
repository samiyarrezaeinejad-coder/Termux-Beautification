# Termux-Beautification
سلام خدمت شما دوستان عزیز.
امروز اومدم با اسکریپت زیباسازی لینوکس و ترموکس
 و توجه کنید این اسکریپت فقط برای ترموکس نیست


 
```
apt update
apt upgrade
pkg install git
git clone https://github.com/samiyarrezaeinejad-coder/Termux-Beautification.git
termux-setup-storage
pkg install python
pkg install pip
python --version
pip --version
python -m ensurepip --upgrade
python -m pip install --upgrade pip
python -m pip install art
nano ~/.bashrc
clear
python3 ~/Termux-Beautification/'termux beautification'
```
