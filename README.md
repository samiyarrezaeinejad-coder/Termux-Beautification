# Termux-Beautification
سلام خدمت شما دوستان عزیز.
امروز اومدم با اسکریپت زیباسازی لینوکس و ترموکس
 و توجه کنید این اسکریپت فقط برای ترموکس نیست
 حتما توضیحاتی که جدا شدن رو بخونید 


 
```
apt update
apt upgrade
pkg install git
pkg install python
pkg install pip
python --version
pip --version
python -m ensurepip --upgrade
python -m pip install --upgrade pip
python -m pip install art
nano ~/.bashhrc
clear
python ~/termux beautification.py
```



دستور ها به صورت تکی جهت توضیح
```
apt update
apt upgrade
```
ابتدا ترموکس خود را اپدیت کنید
```
pkg install git
```
برای باز کردن لینک های گیت هابی نیاز هست و خوب هست که روی لینوکس یا ترموکس نصب باشد
```
pkg install python
```
با این دستور پکیج پایتون رو نصب کنید
```
pkg install pip
```
با این دستور پکیج پی ای پی رو نصب کنید
```
python --version
```
با این دستور برسی کنید که پایتون به درستی نصب شده اگر پیامی مثل ( python 1.13) باشه یعنی درست نصب شده
```
pip --version
```
با این دستور برسی کنید که پی ای به درستی نصب شده اگر پیامی مثل ( pip 1.13 ) باشه یعنی درست نصب شده
```
python -m ensurepip --upgrade
```
این دستور مربوط میشه به اپدیت و ارتقای پکیج پایتون میشه
```
python -m pip install --upgrade pip
```
این دستور مربوط به اپدیت و ارتقای پکیج پی ای پی میشه
```
python -m pip install art
```
این دستور مربوط به نصب پکیج یا ماژول ارت میشه توسط پی ای پی
```
nano ~/.bashhrc
```
با این دستور وارد بخشی میشید که با اجرای هربار ترمینالتون این دستور ها یا فایل ها اجرا بشن
```
clear
```
با این دستور که البته دلخواه هست ترمینالتون با ورود به ترمینال متن های روی صفحه ترمینال حذف میشه و صفحه خالی میشه
```
python ~/termux beautification.py
```
با این دستور که بعد از دستور قبلی نوشته میشه باعث باز شدن فایل پایتونی که ساختیم میشه و اون رو اجرا میکنه.
